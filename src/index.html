Discussion Board

Discussion Board Preview

Technologies Used: Ruby on Rails, JavaScript, jQuery, Bulma.

Discussion Board is an open source website I'm creating for The Flatiron School. I'm using Ruby on Rails, JavaScript, jQuery, and Bulma to create this project.

  • has_many :through: A relationship which creates a join table that allows a user to find a discussion and discussion to find a user. A has_many :through relationship makes everything available, such as categories, or in this case discussions.

  • Gravatar_image_tag: Shows a users gravatar image if they're signed in.

  • Rolify and Cancancan: Helps with an admins user role by hooking into devise to make the admin user role.

  • Friendly_id: Helps create slugs (a slug is a name given to a URL of a page) for our URL's, and is responsible for making the URL an actual slug rather than numbers. Makes it easier to remember and when you share it with someone.

  • Bulma: A simple, elegant, and modern CSS framework that has a different feel to it than Bootstrap.
